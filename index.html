How to use:

  1. Add a process using the input field at the top
  2. For each process, add the required subprocesses
  3. The timer starts automatically when you add the first subprocess
  4. For each subprocess, specify:
    • Activity Type (VA/NVA)
    • Remarks for additional information
    • Number of persons required
  5. When a subprocess is completed, click its "Lap" button
  6. You can record multiple lap readings for the same subprocess
  7. Add a new subprocess when ready to move to the next step
  8. Click "Export to Excel" to save all your recorded data
